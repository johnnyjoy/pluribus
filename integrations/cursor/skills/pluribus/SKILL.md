# Pluribus Skill

## Loop

1. recall_context (before work)
2. plan / act
3. record_experience (after outcome)

## Use When

- Any non-trivial task
- Debugging
- Repeated issues
- Learning new patterns

## Principle

If you do not recall, you risk repeating mistakes.  
If you do not record, you lose what you learned.
