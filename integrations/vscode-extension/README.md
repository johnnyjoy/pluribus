# VS Code extension (alias path)

The **source** for the Pluribus VS Code extension lives at **`../vscode/extension/`** (TypeScript).

CI builds that path and attaches **`pluribus-memory-<tag>.vsix`** to GitHub Releases on version tags `v*`. See [`integrations/vscode/README.md`](../vscode/README.md) and [`BUILD.md`](../../BUILD.md).
